# Sales Performance Dashboard

## Project Overview
This project analyzes retail sales data using SQL and Tableau to identify sales trends, profitable categories, regional performance, and top-selling products.

## Tools Used
- Excel
- MySQL
- Tableau

## Key Features
- Sales KPI Analysis
- Profit KPI Analysis
- Regional Sales Analysis
- Monthly Sales Trend
- Top 10 Products Analysis
- Discount vs Profit Analysis
- Interactive Dashboard

## SQL Analysis Performed
- Total Sales
- Sales by Region
- Profit by Category
- Top Products
- Customer Segment Analysis
- Monthly Sales Trend

## Key Insights
- Technology category generated the highest profit
- West region contributed the highest sales
- Higher discounts negatively impacted profitability
- Certain products generated significantly higher revenue

## Dashboard
Interactive Tableau dashboard created for business performance monitoring and trend analysis.
